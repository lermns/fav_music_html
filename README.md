# Mi Disco Favorito

Página web estática que presenta el álbum favorito del autor: **いたずら天使 (Little Rascal)** de **Toshifumi Hinata** (1995). Incluye fotos del artista, un vídeo y una tabla descriptiva del disco, todo sobre un fondo personalizado.

---

## 📁 Estructura del proyecto

```
├── favMusic.html   # Página principal
├── style.css       # Estilos de la página
└── README.md       # Este archivo (README)
```

---

## 🚀 Cómo usarlo en local

No necesitas instalar nada. Solo sigue estos pasos:

**1. Clona o descarga el repositorio**

```bash
git clone git@github.com:lermns/fav_music_html.git
```

O descarga el ZIP desde GitHub y descomprímelo.

**2. Abre la carpeta del proyecto**

```bash
cd fav_music_html
```

**3. Abre el archivo HTML en tu navegador**

Puedes hacerlo de dos formas:

- Haz doble clic en `favMusic.html` desde el explorador de archivos.
- O desde la terminal:
  ```bash
  # En Windows
  start favMusic.html
  ```

---

## ⚠️ Requisitos

- **Conexión a internet** — las imágenes y el vídeo se cargan desde Cloudinary (hosting externo), por lo que no funcionarán sin conexión.
